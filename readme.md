# Kyber Crystal (series two) RFID Files for Flipper Zero!

So I was told countless times that no, the flipper zero CANT write to kyber crystal Series two chips but low and behold yes, yes you can without any tinkering either!
Simply copy these over to your flipper and use the RFID write tool, it can be a bit finnicky though so make sure you hold the kyber krystal REALLY close to the flipper, thanks!
